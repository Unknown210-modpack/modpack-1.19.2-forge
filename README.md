# How to download Mods?

1. *Click on* the **Green button** that says *Code*.
2. **Scroll Down** to last option.
3. Option is called **Download Zip**.
4. If your Minecraft Launcher does not provide a 1.19.2 Forge mod loader
Run the Jar file outside the mods folder first and install it.
5. When you open your minecraft Launcher choose the **1.19.2 Forge** Game
Version.
6. The rest I am sure you can figure out.
